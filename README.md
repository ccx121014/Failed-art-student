# 落榜美术生（希特勒）模拟器

一款基于AI的人生重开模拟器，模拟阿道夫·希特勒在1907年艺术学院落榜后的人生选择，通过NVIDIA的deepseek-v3_2模型生成多个选择分支，允许玩家改变历史进程。

## 项目简介

本项目是一个基于文本的交互式历史模拟游戏，玩家可以通过不同的选择，探索希特勒在艺术学院落榜后的不同人生轨迹。游戏使用NVIDIA的deepseek-v3_2模型生成选择和结果，提供与真实历史有偏差的可能性，增加游戏的可玩性。

## 功能特点

- 基于AI的交互式历史模拟
- 多个选择分支，影响历史走向
- 详细的结果描述和人生总结
- 与真实历史的对比分析
- 内容过滤和安全机制
- 性能优化和缓存机制

## 技术栈

- Node.js
- Axios (HTTP客户端)
- Dotenv (环境变量管理)
- NVIDIA deepseek-v3_2 (AI模型)

## 项目结构

```
├── index.js              # 主入口文件
├── package.json          # 项目配置和依赖
├── .env                  # 环境变量配置
└── src/
    ├── game/
    │   ├── engine.js     # 游戏引擎
    │   └── scenes.js     # 场景管理
    ├── ai/
    │   └── interface.js  # AI接口
    └── utils/

```

## 安装和运行

1. 克隆项目到本地
2. 安装依赖：`npm install`
3. 配置环境变量：在`.env`文件中设置`DEEPSEEK_API_KEY`
4. 运行游戏：`npm start`

## 游戏玩法

1. 游戏开始时，玩家会看到1907年希特勒艺术学院落榜的背景介绍
2. 系统会生成多个选择分支，玩家需要选择其中一个
3. 根据玩家的选择，系统会生成相应的结果和新的选择
4. 游戏会记录玩家的选择历史，形成完整的人生轨迹
5. 当游戏达到预设的结束条件时，会生成最终的人生总结和历史影响分析

## 注意事项

- 本游戏仅用于娱乐和教育目的，不美化或宣传任何历史人物的负面行为
- 游戏内容经过过滤，确保符合道德和法律标准
- 游戏允许合理的虚构，不追求100%的历史准确性
